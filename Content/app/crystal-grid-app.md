# CrystalGrid

CrystalGrid is a fresh take on Sudoku — designed to be clean, intuitive, and challenging for puzzle lovers of all levels.
The main focus of the current version is to provide a distraction-free Sudoku environment with full support for Apple accessibility features.

## Purpose & Functionality

- Users can play standard 9×9 Sudoku puzzles
- The interface displays a puzzle grid, number input panel, and minimal controls
- Game state can be shared between devices via iCloud and Game Center
- No login, account creation, or network communication is required to play the game

## Accessibility Features

CrystalGrid has been built with accessibility in mind:

- VoiceOver (all puzzle cells, controls, and states are accessible through descriptive labels)
- Voice Control (all buttons and interactive elements are labeled for command-based navigation)
- Dynamic Type / Larger Text (supports larger text sizes for improved readability)
